# OneBitCode – Estudos Full Stack

Repositório destinado aos estudos e práticas desenvolvidas durante a formação
Full Stack, com foco em JavaScript, Node.js e React.

## Objetivo
Consolidar fundamentos, aplicar conceitos na prática e manter um histórico
organizado de evolução técnica.

## Estrutura do repositório
- fundamentos: lógica e conceitos iniciais
- javascript: práticas com JavaScript
- nodejs: backend com Node.js
- react: frontend com React
- projetos: aplicações práticas integrando conceitos

## Metodologia
- Commits frequentes e objetivos
- Organização progressiva
- Código simples e funcional
